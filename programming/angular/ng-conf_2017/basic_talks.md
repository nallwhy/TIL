### Angular CLI

```bash
# Don't write the files, but report them
$ ng new my-app --dry-run

# Generally
$ ng new my-app --routing --prefix my --style scss

# ng serve & open on a browser
$ ng s -o

# Generate component
$ ng g c nav

# Generate service & add to provider
$ ng g s data -m app.module

# Use webpack config
$ ng eject
```n
